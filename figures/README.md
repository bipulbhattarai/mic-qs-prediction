## Overview

Microbiologically influenced corrosion (MIC) causes an estimated \$30–50 billion
in annual infrastructure losses globally. To our knowledge, this is among the first
machine learning frameworks to explicitly integrate quorum sensing (QS) biofilm
community features — alongside environmental and microbial community parameters —
for MIC risk classification.

Using a curated dataset of 78 experimental records compiled from more than 15
published MIC studies, Random Forest achieved:

| Metric | Value |
|--------|-------|
| Cross-validated F1 | **0.762 ± 0.063** |
| AUC-ROC | **0.846** |
| Test accuracy | **81.2%** |

SHAP analysis identified dissolved oxygen, temperature, chemical stress score,
and biofilm community complexity as the top predictors, with QS-derived composite
features contributing to MIC risk stratification.

## Quick Start

```bash
# 1. Clone
git clone https://github.com/bipulbhattarai/mic-qs-prediction.git
cd mic-qs-prediction

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set Anthropic API key (needed for LLM extraction step only)
export ANTHROPIC_API_KEY="sk-ant-..."

# 4. Run full pipeline
chmod +x run_all.sh
./run_all.sh
```

**Estimated runtime:** ~20–30 min (LLM extraction step dominates)
**Estimated API cost:** ~\$0.05–0.10 USD (Claude Haiku)

## Pipeline Steps

| Script | What it does | Runtime |
|--------|-------------|---------|
| `01_fetch_pubmed.py` | Fetch ~650 MIC paper abstracts from PubMed | 3–5 min |
| `02_llm_extract.py` | Extract structured features via Claude Haiku | 10–15 min |

| `03b_build_curated_dataset.py` | Build curated 78-record dataset from key papers | <1 min |

## Dataset

The 78-record dataset was assembled through semi-automated literature extraction
followed by manual verification from more than 15 published MIC studies:

| Parameter | Value |
|-----------|-------|
| Total records | 78 |
| MIC rate range | 0.5 – 2,400 μm/year |
| High MIC (>50 μm/yr) | 53 (68%) |
| Low MIC (≤50 μm/yr) | 25 (32%) |
| Records with QS data | 18 (23%) |
| Metal substrates | Carbon steel (61), SS (14), Cu/Al (3) |
| Environments | Lab (44), Seawater (18), Soil (10), Pipeline (6) |

**Key source papers include:**
Scarascia et al. 2019 (QS + D. vulgaris corrosion),
Jia et al. 2017 (P. aeruginosa MIC),
Xu & Gu 2014 (SRB carbon starvation),
Chen et al. 2021 (X80 soil MIC),
Xu et al. 2024 (13Cr pipeline MIC),
Permeh et al. 2021 (marine crevice MIC)

## Feature Categories

| Category | Features | Role |
|----------|---------|------|
| **QS features** ★ | qs_activity_score, ahl_mentioned, qs_biofilm_synergy | Novel contribution |
| **Biofilm community** | srb_present, biofilm_aggression_score, community_complexity | Core MIC drivers |
| **Environmental** | do_mg_l, temperature_c, chem_stress_score, sulfate, h2s | Physical conditions |

## Results

### Model Performance

| Model | Accuracy | F1 (CV) | F1 (Test) | AUC |
|-------|----------|---------|-----------|-----|
| **Random Forest** | **0.812** | **0.762±0.063** | **0.806** | **0.846** |
| XGBoost | 0.812 | 0.690±0.131 | 0.806 | 0.754 |
| SVM | 0.812 | 0.738±0.104 | 0.806 | 0.773 |
| MLP | 0.688 | 0.573±0.053 | 0.560 | 0.882 |

### Top SHAP Features

1. `do_mg_l` — dissolved oxygen (0.347)
2. `temperature_c` — temperature (0.301)
3. `chem_stress_score` — chemical stress composite (0.235)
4. `sulfate_mg_l` — sulfate concentration (0.225)
5. `community_complexity` — microbial community diversity (0.151)

QS-derived features (`qs_biofilm_synergy`, `biofilm_aggression_score`) rank in top 10.
